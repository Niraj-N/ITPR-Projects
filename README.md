# 💰 Smart Expense Tracker

A Java console-based expense management application developed using Java, JDBC, and MySQL.

## 🚀 Features

- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- MySQL Database Integration
- JDBC Connectivity

## 🛠 Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE
- Git & GitHub

## 📁 Project Structure

```
src/
├── dao/
├── database/
├── main/
└── model/
```
